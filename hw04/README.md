# proc_observer.sh. Описание. Инструменты, использованные в ходе работы. Запуск
proc_observer.sh - это программа, которая находит все процессы, либо же находит процессы по имени
которое мы задали, после чего выводит информацию о количестве процессов, занимаемые ими ресурсы и память,
после чего берет самый первый PID в списке и выводит количество его FD, путь, имя и состояние(например: sleeping).
Сам код разделен на 4 секции и 5 функций - find_processes()`, `show_summary()`, `show_fd()`, `show_proc_info()` и `main` -
где выполняется весь код

Далее вы увидите пример выполненного кода.

Использование: ./proc_observer.sh <имя процесса>

для всех процессов, что содержат "<название процесса>"
ps -eo | grep "$1" | grep -v grep

для суммарного %CPU и %MEM
awk '{sum += $N} END {print sum}

для первого PID найти число файловых дескрипторов:
ls -la /proc/$PID/fd/ 2>/dev/null

для резидентной памяти, состоянии процесса и его имени:
cat /proc/$PID/comm
cat /proc/$PID/status | grep State
cat /proc/$PID/status | grep VmRSS
