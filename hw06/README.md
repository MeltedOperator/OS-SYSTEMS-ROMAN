# Описание и инструкция по применению scheduler_experiment.sh
scheduler_experiment.sh это программа, которая создает два процесса и несколько потоков, получает их
PID, дает им приоритетную позицию nice(-10 и 19), после чего сравнивает скорость их выполнения.
Эксперимент наглядно показывает разницу между процессами с разными приоритетами при загруженных дисках

Запуск программы:
bash $ sudo ./scheduler_experiment.sh <имя директории где лежит README.md>

без sudo программа не запустится потому что nice -n нужны права доступа владельца - то есть вас,
НЕ обычных пользователей для использования ресурсов ядер.

# Пример вывода scheduler_experiment.sh
=== nice 19 ===

real    0m 22.145s
user    0m 0.007s
sys     0m 0.112s

=== nice -10 ===

real    0m 0.097s
user    0m 0.000s
sys     0m 0.088s
